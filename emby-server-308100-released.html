10/12/2016

Emby Server 3.0.8100 Released

New Single Click Recordings

 

Want to record something? Just toggle it. It's as easy as a single click. No more filling out a form to configure the recording.

 

 

Of course you can still configure it. Just click the Settings button after scheduling.

 

 

Time-shift your Live TV !

 

You can now pause, rewind, and fast-forward your Live TV. This feature is now available in the Emby Web app using Chrome, Firefox, Opera, and Microsoft Edge. We've begun our support for this feature in the web app, but look for it in other apps soon.

 

Live TV Stream Sharing

 

If multiple devices play the same channel, Emby can now share a single stream between all of them, keeping your tuners as free as possible. This also allows for more flexible recording padding without creating a conflict.

 

Play In-Progress Recordings

 

Not only can you play your in-progress recordings, but you can seek as well. No more having to wait for the recording to finish.

 

 

New and Improved Recording Designs

 

The screens are now much more graphical and colorful.

 

 

New Series Recording Management

 

Managing series recordings is a snap, and is now a much more graphical process. We think you'll enjoy it.

 

 

 

New Guide Features

 

The guide now requires less clicks thanks to the new popup that allows you to quickly play, schedule a recording, or just read the overview.

 

 

Upcoming on TV for Actors

 

Now when you click on an actor and view their detail screen, you'll now be able to see what upcoming shows they'll be appearing in.

 

 

Have Some Color with Your Posters

 

On some devices, you'll notice more color attached to your posters. This really helps bring the app to life and makes it feel more your own, as the colors are based on your artwork.

 

 

 

No More Path Substitution

 

When you add media folders, you now have the ability to specify the corresponding network path. This is now our replacement for path substitution. Specifying the network path can allow Emby apps on other devices to access your media files directly, which can sometimes lead to a better playback experience.

 

 

Other Improvements

  • Rework upnp discovery
  • Update German ratings for parental control
  • Improve handling against guide changing for a scheduled recording
  • Don't add year to series recording folder paths
  • Exclude unrated trailers when parental control setting is applied
  • Update sqlite for windows
  • Normalize german movie db ratings from de- to fsk-
  • Normalize subtitle codec names
  • Improve prioritization of metadata results by language
  • Ignore #recycle folder for synology
  • Allow adding channel items to collections and playlists
  • Support more raw image formats

 

Bug Fixes

  • Add library monitor error handling
  • Fix recorded episodes being detected as movies
  • Exclude unrated trailers when parental control setting is applied
  • Fix xml tv movie/series detection
  • Fix layout of block by tags
  • Fix repeated refreshing from realtime monitor
